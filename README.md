# Love Calculator

## Introduction

The Love Calculator is a fun and lighthearted web-based tool that generates a random love percentage based on the names entered.

## How to Use

1. **Enter Names:**
   - Input the names of two individuals in the provided text boxes.

2. **Calculate Love Percentage:**
   - Click on the "Calculate Love Percentage" button to generate a random love score.

3. **Interpret Results:**
   - The calculator will display a love percentage along with a playful message.

## Love Percentage Range

- **0% - 20%:** Low Compatibility
- **21% - 50%:** Moderate Compatibility
- **51% - 80%:** High Compatibility
- **81% - 100%:** True Love

## Algorithm Used

The Love Calculator uses a random algorithm to generate love percentages. It's designed for entertainment purposes and is not based on any scientific or astrological principles.

To use this Love Calculator, simply open the [Love Calculator](#) in your web browser.
